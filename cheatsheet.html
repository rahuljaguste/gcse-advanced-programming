📁 File Handling

"w" writeCreates/overwrites file
"a" appendAdds to end (safe!)
"r" readRead contents
# Write
f = open("data.txt", "a")
f.write("Hello\n")
f.close()

# Read
f = open("data.txt", "r")
lines = f.readlines()  # list
f.close()

Always close files! "w" erases first — use "a" to be safe.

🔍 Searching Algorithms

Linear searchCheck each item, one by one
Sorted needed?No
Worst caseN comparisons
Binary searchCheck middle, discard half, repeat
Sorted needed?Yes!
Worst caselog₂(N) comparisons
def linear_search(data, target):
    for i in range(len(data)):
        if data[i] == target:
            return i
    return -1

def binary_search(data, target):
    low, high = 0, len(data) - 1
    while low <= high:
        mid = (low + high) // 2
        if data[mid] == target: return mid
        elif data[mid] < target: low = mid + 1
        else: high = mid - 1
    return -1

🔀 Sorting Algorithms

Bubble sortCompare neighbours, swap, repeat
SpeedSlow — N² comparisons
MemoryIn-place (no extra)
Merge sortSplit in half, sort each, merge back
SpeedFast — N × log₂(N)
MemoryUses extra (new lists)
def bubble_sort(data):
    for p in range(len(data) - 1):
        for i in range(len(data) - 1 - p):
            if data[i] > data[i+1]:
                data[i], data[i+1] = data[i+1], data[i]

1,000 items: Bubble ≈ 1,000,000 ops | Merge ≈ 10,000 ops
